호흡역학 · 폐보호환기 · 동맥혈가스 — 이론을 손으로 만진다
🔍
10개 토픽 · 각 토픽은 핵심 → 쉽게 이해하기 → 직접 해보기 → 더 깊이 → 핵심정리 → 함정 → 출처 순으로 펼쳐집니다.
실제 환자에서 규칙들이 충돌하는 한계 상황. CO₂를 맞추면 SpO₂가 무너지고, 산소를 살리면 압력이 폐를 찢는 — 무엇을 포기할지의 줄타기.
빼는 것도 다는 것만큼 기술이다. 환자가 깨어 셀프가 살아나도 얼렁뚱땅 빼면 30%가 다시 들어간다. 지켜야 할 절차와 챙겨야 할 관문.
모든 수치·공식은 출처에 근거하며 각 항목 하단에 링크를 답니다. 이 도구는 교육·의사결정 보조용이며 실제 처방을 대체하지 않습니다. 극단 상황의 허용 경계는 환자 개별 상태(혈역학·동반질환)에 따라 달라지며, 제시된 숫자는 출발점일 뿐입니다.

🫁 폐역학 시뮬레이터

질환을 고르고 노브를 돌려 압력·파형·auto-PEEP을 실시간으로 본다.

⚠️ 단순화 교육 모델 — 폐를 단일 구획(균질 R·C)으로 가정합니다. 실제 COPD·ARDS는 부위마다 시간상수가 다른 불균질 폐(2구획 이상)이고, PEEP에 따른 recruitment(폐포 열림)나 흡기정지 시 실제 Pplat도 이보다 복잡합니다. 개념의 방향을 익히는 도구이며, 실제 환자의 절대 수치를 대체하지 않습니다.
